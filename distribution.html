Categorical Logic · Terms

Understanding
Distribution

A term is distributed in a proposition when that proposition makes a claim about every member of the class the term refers to. Knowing which terms are distributed is essential for evaluating syllogisms.

Distributed
The proposition says something about all members of that class — every single one is covered by the claim. The term's entire class is referenced.
Undistributed
The proposition only says something about some (or an unspecified portion) of that class — not every member is covered. The term's entire class is not referenced.
Filled = Distributed (whole class is claimed)
Dashed = Undistributed (partial claim only)

Two Rules to Remember It All
S
Universals distribute their subjects
Particulars do not.
A and E make claims about all members of S — so S is distributed. I and O only talk about some — so S is not.
P
Negatives distribute their predicates
Affirmatives do not.
E and O exclude something from the entire predicate class — so P is distributed. A and I only affirm partial membership — so P is not.
